Mod Information
=======

Factorial is a tech mod that aims to differentiate itself from the others by focusing on accessibility, quality-over-quantity, and not overcomplicating things.

Content
============
This mod aims to provide a clear way to automate every resource, and provide easy scalability through upgradeable machines.

Every machine and power generator should be able to be made at roughly the same time as every other machine, so no gatekeeping resource automation behind five levels of tech machines and power generators.
